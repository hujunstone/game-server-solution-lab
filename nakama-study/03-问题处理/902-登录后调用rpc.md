2. 用玩家 Session Token 调 RPC
Authorization 选择 Bearer 方式，填入 token 
http post 方式
http://192.168.50.66:7350/v2/rpc/demo_faction_get?unwrap=true
- **Auth**：No Auth（不要 Basic）。
- **Headers**：
    - `Content-Type: application/json`
    - `Authorization: Bearer <SESSION_TOKEN>`（只保留这一种 Authorization）。
- **Body**：
```json
{
"code": "JINGSHANG"
}
```
