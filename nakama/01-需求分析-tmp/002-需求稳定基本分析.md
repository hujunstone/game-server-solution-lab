### 3. 身份与属性
- **势力声望等级**：无名 → 知名 → 尊敬 → 崇敬 → 传奇（需晋商势力专属Icon）。
- **核心属性**：力量、敏捷、体质、智慧（初始各10，出身再分配，总点数40）。
- **基础公式**：  
  - HP = 体质×10 + 力量×2  
  - DEF = 体质×3 + 力量×1  
  - 近战伤害 = 基础伤害 + 力量 + 武器  
  - 远程伤害 = 基础伤害 + 敏捷

（出身差异：秀才偏智，趟子偏力，牙行偏敏，数值已给出）

### 4. 技能系统
- 技能分主动/被动，可升级至10级。

**商贾之道**
- 识货（被动）：提升购买量  
  - 提升 = 3% + 智慧×0.2%
- 议价（主动）：影响买入/卖出价  
  - 浮动 = 智慧×0.1% + 技能等级×0.5%
- 打探（被动）：获取情报  
  - 成功率 = 15% + 智慧×1% + 技能等级×3%

**防卫之术**
- 攻击：无CD普攻  
  - 伤害 = 40 + 力量 + 武器
- 重劈（单体高伤）：  
  - CD：3/2/1回合（随等级段降低）  
  - 伤害 = (40 + 力量)×(1 + 技能等级×0.1)  
  - 击晕概率 = 10% + 技能等级×2%
- 破甲：下降敌人防御  
  - 伤害 = (40 + 力量)×0.9  
  - 降低DEF = 8 + 技能等级×2，持续3回合  
  - CD：4/3/2回合（随等级段降低）
- 包扎：无CD治疗，清除流血  
  - 治疗 = 15 + 技能等级×2
- 治疗：消耗1金疮药，CD 3回合  
  - 恢复 = 30 + 技能等级×4
- 手术：消耗1麻沸散，CD 5回合  
  - 恢复 = 50 + 技能等级×5；若HP<30%，额外回复缺失生命30%。

## 二、战斗系统（补充部分）

### 8. 多队战役结构（补充说明）
- **适用场景**：  
  - 扫荡云中寨等山寨；  
  - 争夺贸易点控制权（如关口、商栈据点）。
- **流程概括**：  
  1. **大地图阶段**：  
     - 玩家控制多支小队在区域地图上机动；  
     - 行军、包围、卡位等行为以即时或战棋方式呈现。  
  2. **小队战斗阶段（回合制）**：  
     - 任意两支敌对队伍接触即切入当前已有的回合制战斗系统；  
     - 战斗胜负决定该队伍是否撤退、全灭或继续推进。  
  3. **建筑攻击阶段（据点耐久）**：  
     - 攻打城门、寨门、大殿等时按时间/轮数扣除建筑耐久；  
     - 若攻方被敌军骚扰，则攻城被中断并进入小队战斗。  
  4. **胜负条件**：  
     - 扫荡山寨：击杀寨主/BOSS；  
     - 争夺贸易点：攻方夺旗成功，守方阻止一定时间即防守成功。

## 三、跑商系统（完整精简）

### 1. 货物系统

#### 1.1 商品系统
**商品品类：**  
米粮、杂食、布帛、皮货、铁器、木器、瓷器、漆器、药材、酒、调料、杂货、茶、香料、番货、盐。  
每类下有典型商品及基础价格与稀有度（★）。

**示例：**  
- 米粮：小麦、高粱（60文/30件）。  
- 铁器：铁锅(500文/10件)，大阳钢针(700文/4件，★★★)。  
- 布帛：潞绸(4000文/3件，★★★★)。  
- 酒：汾酒(1200文/8件，★★★)。

**贸易点商品示例：**  
- 太原府：小麦、高粱、汾酒、陈醋、毛毡、粗布、野核桃。  
- 大同：胡麻、熟皮、皮靴、皮袄、皮帽。

#### 1.2 商品价格模型

**核心变量：**  
- `BasePrice`：基准价  
- `Development`：城市发展度  
- `HomePreference`：本地偏好  
- `Reputation`：玩家声望  
- `Rarity`：稀有度  
- `DemandShock` / `SupplyShock`：供需波动  
- `GlobalFactor`：全局平衡系数  
- `TaxMarkup`：税差

**计算：**  
1. 价格基准系数：  
   `H_D = (1 + k_p * HomePreference) / (1 + k_d * Development)`  

2. 本地售价：  
   `SalePrice = BasePrice * (1 + TaxMarkup) * H_D * GlobalFactor * DemandShock`  

3. 本地收购价：  
   `PurchasePrice = BasePrice * (1 - TaxMarkup) * H_D * GlobalFactor * SupplyShock`  

4. 玩家可购配额：  
   `Quota = (BaseQuota + k_d * Development) * (1 + k_r * Reputation) * Rarity * (1 / GlobalFactor)`

#### 1.3 刷新机制
- 配额每 15 分钟刷新一次，可受事件影响波动。

### 3. 贸易点系统

#### 3.1 贸易点与商路

* **主要贸易点/节点**
  * 太原府 → 石岭关 → 雁门关 → 大同府  
  * 特殊地点：云中寨（山寨据点）、雁门关外（可建商栈）

* **商路风险（基础值）**
  * 太原 ↔ 石岭关：风险 **0**
  * 石岭关 ↔ 雁门关：风险 **30%**（山匪/流民）  
    * 扫荡云中寨后降至 **5%**
  * 雁门关 ↔ 大同府：风险 **50%**（马贼）  
    * 雁门关外建立商栈后降至 **20%**，并可在遭遇战时**呼叫支援**

#### 3.2 区域划分与关键NPC

* **大地图区域**
  1. 太原府区域（石岭关以南）
  2. 关隘通道（石岭关–雁门关）
  3. 大同府区域（雁门关以北）

* **太原府**
  * 晋王府：老承奉、仪卫百户  
  * 三司署衙：经历王秉忠（路引相关）、老书办（官府任务）  
  * 市集：牙行人赵四（交易）、刘婆（打探消息）  
  * 晋商总会：管事乔圣才（晋商势力相关）

* **大同府**
  * 总兵府：旗牌官赵铁山（边贸相关）  
  * 兵备道衙门：书吏吴九思（开中法相关）  
  * 作坊区：皮匠老鄂（委托生产）

* **云中寨**
  * 场景：山门陷坑、聚义厅、兵器库  
  * 关键NPC：守门喽啰（战斗/任务入口）
