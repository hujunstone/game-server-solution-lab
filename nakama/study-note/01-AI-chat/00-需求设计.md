### 2.3 技能系统
技能分为主动技能和被动技能，以及可升级技能，最高为10级。
商贾之道
│
└─ 识货 ─ 专精
│
└─ 议价 ─ 牙行通
│
└─ 打探 ─ 行情

识货：被动技能，提升购买量。 购买量提升 = 基础值（3%） + （智慧 × 0.2）%
议价：主动技能，影响商品的买入价和卖出价
价格浮动 = (智慧×0.1%  + 技能等级×0.5%)
比如智慧15，技能等级5级的商人，价格浮动为 (1.5% + 2.5%) = 4%
打探：被动技能，从NPC口中获取有用信息
打探成功率 = 基础成功率15% + 智慧 × 1% + 技能等级 × 3%

防卫之术
│
└─ 攻击 ─ 重劈 ─ 破甲
│
└─ 包扎 ─ 治疗 ─ 手术

攻击：主动技能，普通攻击。无冷却
伤害：基础伤害40 + 力量 × 1 + 武器伤害值
重劈：主动技能，高伤害的单体攻击。冷却回合数Lv0-2:3 / Lv3-6:2 / Lv7-10:1
伤害：(基础伤害40 + 力量 × 1) × (1.0 + 技能等级 × 0.1)
额外效果：有概率击晕敌人，击晕概率 = 10% + 技能等级 × 2%
破甲：对目标发动一次精准的打击，旨在破坏其护甲。施加一个强力的减益效果。
伤害：破甲伤害 = (基础伤害40 + 力量 × 1) × 0.9
效果公式：
效果：使目标的防御力降低。
破甲量：降低目标 = 8 + 技能等级 × 2 点 的防御力。
持续时间：3 回合。
冷却时间：Lv0-3:4 / Lv4-7:3 / Lv8-10:2

包扎：主动技能，无冷却
      治疗 = 15 + 技能等级 × 2
并清除【流血】状态
治疗：恢复 30 + 技能等级 × 4
消耗 1份金疮药
冷却：3回合
手术：恢复 50 + 技能等级 × 5 点生命值；若目标HP<30%，额外恢复缺失生命值的30%
      消耗一个麻沸散
冷却：5回合

技能	普攻基础伤	技能伤害	DEF=40 实际伤	额外效果
普攻	40+15=55	55	15	—
重劈	55	55×(1.0+0.5)=82.5	42.5	20%概率击晕
破甲	55	55 × 0.9 = 49.5	9.5	降防=8+10=18点→ 敌人DEF从40→22
下回合普攻实际伤害 = 55 − 22 = 33（+120%提升！）

### 3.3 核心战斗流程
每回合分为 3阶段：
1.速度判定（先攻顺序）
计算公式：出手速度 = 敏捷 × 0.8 + 鞋子加成 + 随机值(−2~+2)
敌我混合排序（快者先动）
2.行动选择
可选：
(1)普通攻击
(2)技能（重劈 / 破甲 / 治疗 / 包扎 / 手术）
(3)使用道具（金疮药、烟雾弹）
(4)战术指令
3.状态结算
(1)流血、中毒等持续伤害
(2)破甲效果衰减
(3)击晕单位跳过下回合
### 3.4 防御与命中机制
命中率 = 50% + (玩家敏捷 − 敌人敏捷) × 0.5%
基础50%，敏捷差20点 = ±10%命中
最低30%，最高80%（防极端）
伤害公式
实际伤害 = max(3,  (40 + 力量 + 武器伤害) − 敌人DEF)
特殊效果触发（如击晕）：
先判定命中 → 再判定特效概率
例：重劈 LV.5 → 击晕概率 = 20%
若未命中，则特效不触发

### 3.6 多团队战役（扫荡山寨 以及 争夺贸易点）
基本策略是 即时战略地图 + 回合制战斗
1.大地图阶段（即时制/战棋制）
模式：所有队伍在一个大的战场地图上同时、自由移动。
目的：进行战略调度——调兵遣将、占领要地、包围城门。
交互：移动、集结、使用战场技能（如加速、建造临时路障）。
2.小队战斗阶段（回合制）
触发：当一支队伍的攻击指令目标是一支敌方队伍时（而不是建筑），双方立即进入一个独立的战斗场景。
模式：在此场景内，进行一场完全独立的回合制战斗。
结果：战斗结束后，胜负方被传回大地图。败方队伍可能会被击退
3.建筑攻击阶段（即时计算）
触发：当一支队伍的攻击指令目标是城门/大殿等建筑时。
模式：不进入回合制战斗场景。队伍会立刻开始以一定频率自动攻击建筑，建筑耐久度会持续下降。
中断：如果攻击建筑的队伍在大地图阶段被敌方队伍攻击，则建筑攻击行为会立即中断，并进入小队战斗阶段。
4.最终胜负判定
扫荡山寨以成功击杀BOSS为最终胜利标准
争夺贸易点战斗是以攻击方夺旗为胜利，夺旗失败守卫方胜利

### 4.2 商品价格计算模型
变量	符号	描述
商品基准价​	BasePrice	商品的标准锚定价格，是所有计算的起点。
城市综合发展度​	D	该城市整体的经济繁荣水平。由全体玩家投资行为共同塑造。值越高，城市越发达。​
商品本地偏好度​	D_local	该城市对这种商品的固有需求程度。例如：沙漠城市对“水”的偏好度极高。这是一个固定属性。
玩家声望等级​	R	玩家在该城市的声望等级（如0-10级）。影响可购买量，不影响单价。
商品稀有度​	Rarity	商品本身的稀缺性，影响可购买量。普通=1，稀有=0.5，珍贵=0.2。
即时需求波动​	DemandShock	随机任务或事件导致的临时需求变化乘数（默认1.0）。
即时供应波动​	SupplyShock	随机事件导致的临时供应变化乘数（默认1.0）。
全局供需系数​	GlobalFactor	基于全服该商品近期交易总量计算的动态平衡系数（默认1.0，范围可0.5-1.5）。
交易税差​	Tax_Markup	核心安全系数。代表NPC交易的固定利润率，确保原地买卖亏损。例如0.15（15%）。
发展度影响系数​	k_d	调节城市发展度D对价格影响的强度。例如0.01。
偏好度影响系数​	k_d_local	调节本地偏好度D_local对价格影响的强度。例如0.1。
声望影响系数​	k_r	调节玩家声望R对购买量加成的强度。例如0.15。


1. 价格基准系数（H_D）这个系数综合了城市发展度和本地偏好，决定了该城市对该商品的“合理”价格水平。
H_D = (1.0 + k_d_local * D_local) / (1.0 + k_d * D)
分子代表需求拉动（偏好度越高，基准越高），分母代表供给压制（发展度越高，基准越低）。H_D是后续计算的基础。
2. 商品的本地售价（玩家从城市购买的单位价格）
SalePrice = BasePrice * (1.0 + Tax_Markup) * H_D * GlobalFactor * DemandShock
3. 商品的本地收购价（玩家向城市出售的单位价格）
PurchasePrice = BasePrice * (1.0 - Tax_Markup) * H_D * GlobalFactor * SupplyShock
4. 玩家单种商品可购买量（配额）
AvailableQuota = (BaseQuota + k_q * D) * (1.0 + k_r * R) * Rarity * (1.0 / GlobalFactor)
使用 (1.0 / GlobalFactor)作为GlobalScarcity。当全球热卖（GlobalFactor> 1.0）时，商品更稀缺，配额降低；当商品滞销（GlobalFactor< 1.0）时，配额增加。
